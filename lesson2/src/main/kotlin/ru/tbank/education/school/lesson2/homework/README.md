## Университет
Главный класс, хранит в себе и управляет всеми существующими сущностями<br>
Также хранит в себе экзамены<br>
Может добавлять и удалять любые сущности или экзамены<br>
Может возвращать список сущностей по enum из Roles
### Сущности
Реализован родительский абстрактный класс Person, а также дочерние классы Student и Teacher.<br>
Student: <br>
1. Принимает участие в экзамене, а позже завершает его
2. Узнает и хранит оценку за различные экзамены
<br>Teacher:


1. Создает экзамены
- Задает структуру, описание
- Задает вопросы для экзамена
2. Оценивает студентов по каждому экзамену
3. Удаляет экзамены, принадлежащие ему
4. Получает студентов для оценивания

### Процесс обучения

#### Статусы экзамена
Каждый студент имеет свои стадии прохождения экзамена
1. Не участвует (NotParticipating)
2. Участвует (Participating)
3. Завершил (Finished)
4. Оценен (Rated)
#### Экзамен
1. Хранит в себе учителя, оценки студентов, вопросы
2. Выдает оценку по студенту
3. Добавляет вопросы к экзамену
4. Может присоединить студента к экзамену

#### Вопрос
Дата класс, хранит в себе вопрос и ответ на него
